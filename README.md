## ⌨ How to execute this project:

```bash
# Clone this repository
git clone https://github.com/mohamedstt/studyplatform

# Open the directory
cd studyplatform

# Dowload dependencies
npm i

# Execute the server
npm run dev
```

Done? Open your browser with: `http://localhost:5500/`
